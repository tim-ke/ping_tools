# ping_tools
ping小工具

需求:電腦需要安裝java8(含)以上
適用平台:具有GUI介面，Windows、Mac apple、Linux

解決功能:
1.更換Mac mini後沒有可以同時ping多工具功能
2.原本的ping工具無法測試windows預設情況下聯通性，增加TCP_port
3.測試新開機客戶IP是否有正常
